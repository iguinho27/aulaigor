# Quebra de Linha

## Tag

```
<br>
```

## Conceito

Insere uma quebra de linha dentro do mesmo bloco de texto.

Não cria um novo parágrafo.

---

## Exemplo

```
<p>Rua das Flores, 100<br>Centro<br>Fortaleza - CE</p>
```

Resultado:

```
Rua das Flores, 100CentroFortaleza - CE
```

---

## Quando usar

Situações apropriadas:

- Endereços
- Poemas
- Letras de música
- Informações que exigem quebra visual

---

## Quando NÃO usar

❌ Errado

```
Texto 1<br><br><br>Texto 2
```

Para espaçamento visual utilize CSS.
