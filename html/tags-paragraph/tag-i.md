## Italic

### Tag

```
<i></i>
```

### Conceito

Aplica itálico visualmente.

---

### Exemplo

```
<p><i>Termo estrangeiro</i></p>
```

---

### Uso comum

- Termos técnicos
- Palavras estrangeiras
- Pensamentos
- Nomes científicos

---

## Comparação

```
<em>Ênfase semântica</em><i>Estilo visual</i>
```
