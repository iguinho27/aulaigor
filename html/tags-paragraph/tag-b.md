## Bold

### Tag

```
<b></b>
```

### Conceito

Aplica negrito visualmente.

---

### Exemplo

```
<p><b>Texto em negrito.</b></p>
```

---

### Semântica

Não adiciona importância semântica.

Serve apenas para destaque visual.

---

## Comparação

```
<strong>Importância</strong><b>Apenas visual</b>
```

Prefira `<strong>` quando o conteúdo possuir significado.
