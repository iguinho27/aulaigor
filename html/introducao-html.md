### O que é HTML?

- O significado de HTML (HyperText Markup Language)
- Diferença entre HTML, CSS e JavaScript
- Como o navegador interpreta HTML

O HTML não serve apenas para exibir conteúdo visualmente.

Cada tag possui um significado semântico, ou seja, informa ao navegador, leitores de tela e mecanismos de busca qual é a função daquele conteúdo.

### Ferramentas

- VS Code
- Navegador
- Chrome DevTools

### Primeiro documento HTML

```
<!DOCTYPE html>
<html>
	<head>
		<title>Minha Página</title>
	</head>
	<body>
		<h1>Olá Mundo</h1>
	</body>
</html>
```

## Tags obrigatórias

### `<!DOCTYPE html>`

Define a versão do HTML.

### `<html>`

Elemento raiz da página.

### `<head>`

Metadados da página.

### `<body>`

Conteúdo visível.

---

## Metadados importantes

### `<title>`

Título da aba.

### `<meta>`

```
<meta charset="UTF-8"><meta name="viewport" content="width=device-width, initial-scale=1.0">
```

### `<link>`

Importação de CSS.

### `<script>`

Importação de JavaScript.
